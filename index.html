Quick Installation

Get started in minutes with our streamlined installation process

Linux Installation 
# Clone the repository
git clone https://github.com/teotiaabhi/gun-wifi-tool.git
cd gun-wifi-tool

# Install dependencies
pip install -r requirements.txt

# Run the tool
sudo python3 src/gun_wifi_tool.py --help
System Requirements 
# Install system dependencies (Ubuntu/Debian)
sudo apt update
sudo apt install hostapd dnsmasq aircrack-ng

# Install Python packages
pip install scapy flask qrcode[pil]

# Set WiFi adapter to monitor mode
sudo airmon-ng start wlan0
Kali Linux Installation 
# Kali Linux has most tools pre-installed
git clone https://github.com/teotiaabhi/gun-wifi-tool.git
cd gun-wifi-tool

# Install Python dependencies
pip install -r requirements.txt

# Ready to use!
sudo python3 src/gun_wifi_tool.py --scan
